# Lotto Number Generator

## Overview

A simple web application to generate South Korean lottery numbers. The application will display 6 unique numbers from 1 to 45.

## Project Outline

### Style and Design
*   **Layout:** Centered layout with a clean and modern design.
*   **Colors:** A simple color palette with a primary color for the button and accents.
*   **Typography:** Clear and readable fonts.
*   **Iconography:**  A lottery-related icon could be used to enhance the user experience.

### Features
*   **Number Generation:** Generate 6 unique random numbers between 1 and 45.
*   **Display:** Display the generated numbers in a visually appealing way.
*   **User Interaction:** A button to trigger the number generation.
